# Healthcare Analytics Dashboard

**Live Demo:** [Click here to view the live dashboard](https://healthcaredashboard-nehafilvke3ai64e8wccv5.streamlit.app/)

## Project Overview
This project is an interactive data visualization tool built with Python and Streamlit. It is designed to track, analyze, and display key hospital metrics, specifically focusing on patient wait times. The dashboard processes static healthcare datasets to monitor operational efficiency and highlight critical delays.

## Features
* **Data Processing:** Automated categorization of patient wait times (Critical, Moderate, On Time) using Pandas.
* **Interactive Visualizations:** Dynamic bar charts generated with Matplotlib to display top patient wait times.
* **Data Tables:** Clear, formatted data tables presenting processed metrics for quick review.
* **Cloud-Ready Data:** Utilizes extracted CSV datasets (`hospital_data.csv`) for seamless deployment without requiring live database connections.

## Technologies Used
* **Language:** Python
* **Framework:** Streamlit
* **Data Manipulation:** Pandas
* **Data Visualization:** Matplotlib

## Local Installation

To run this dashboard locally, follow these steps:

**1. Clone the repository**
git clone [https://github.com/Kolek88/healthcare-dashboard.git](https://github.com/Kolek88/healthcare-dashboard.git)
cd healthcare-dashboard

**2. Install dependencies**
pip install -r requirements.txt

**3. Run the application**
streamlit run hospital_app.py
